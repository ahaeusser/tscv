# Seasonal naive model

Automatic train a seasonal naive model (SNAIVE2)..

## Usage

``` r
SNAIVE2(formula, ...)
```

## Arguments

- formula:

  Model specification (see "Specials" section, currently not in use
  ...).

- ...:

  Further arguments.

## Value

snaive2_model An object of class `SNAIVE2`.
