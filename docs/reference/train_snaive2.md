# Seasonal naive model

Train a seasonal naive model (SNAIVE2).

## Usage

``` r
train_snaive2(.data, specials, ...)
```

## Arguments

- .data:

  Input data as tsibble.

- specials:

  Specials as list defined in `specials_snaive2`.

- ...:

  Currently not in use.

## Value

An object of class `SNAIVE2`.
