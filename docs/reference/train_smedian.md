# Seasonal median model

Train seasonal median model (SMEDIAN).

## Usage

``` r
train_smedian(.data, specials, ...)
```

## Arguments

- .data:

  Input data as tsibble.

- specials:

  Specials as list defined in `specials_smedian`.

- ...:

  Currently not in use.

## Value

An object of class `SMEDIAN`.
